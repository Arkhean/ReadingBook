# ReadingBook

Application React Native de gestion de bibliothèque de lecture.


Ajouter des livres à votre bibliothèque en renseignant :
- Titre
- Auteur
- Saga
- Numéro du Tome
- Genre
- Editeur
- Format
- Prix
- Nombre de pages
- Date d'achat
- Dates de lecture
- Commentaire personnel

## Fonctionnalités

- input text
- date picker
- list picker
- animations
- navigation
- redux
- ...
